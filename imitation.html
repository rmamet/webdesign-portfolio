Imitation Project

Web Design II Project

Originally submitted on Sept. 16, 2025

The Imitation Project was a simple project to see what I knew about the design part of web design. The point was to find a design made by a professional designer, and imitate it using HTML and CSS. I used a Shakespear graphic, and made it almost exactally pixel-for-pixel using grid and flex.
